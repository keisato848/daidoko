## 1. ブランド表記

| 用途                 | 表記                           |
| -------------------- | ------------------------------ |
| 正式名称（漢字）     | 臺所                           |
| 正式名称（ひらがな） | だいどこ                       |
| **アプリ内表示名**   | **DAIDOKO**                    |
| サブコピー           | 家族で育てる、台所のレシピ手帳 |

**表記方針:**

- **ロゴ・アイコン・スプラッシュ**: 視覚的なブランドマークとして **臺所**（正字体）を使用する。正字体の複雑な字形が「深夜の私設レシピ蔵書庫」のコンセプトに合致し、格調ある印象を与える。
- **アプリ内 UI テキスト（画面タイトル・ナビゲーション等）**: **DAIDOKO**（アルファベット大文字）を使用する。ホーム画面ではヘッダーバーを廃止してコンテンツ領域を最大化しており、アプリ名は Cormorant Garamond Italic・極小サイズ（9px）でフィルタタブ右端にのみ添える。
- **ひらがな「だいどこ」**: ストア説明文・マーケティング文章でのよみがな表記に限定。UI には使用しない。

## 4. コンセプト一覧

### A · 蔵書印（推奨）

円形の印章スタイル。「私設蔵書庫」「家族の蔵書印」という世界観に直結する。  
アプリアイコン・スプラッシュ・ヘッダー全てに転用可能。

**SVG ファイル:** `docs/brand/svg/logo-seal.svg`

- 外円: stroke `#C9A16A`、1px
- 内円（破線）: stroke `#2E2418`、0.5px、dasharray `3 3`
- 中央テキスト: 臺所（40px 明朝）、fill `#DCC9A8`
- 下部ライン + ラテン文字: `D A I D O K O`（Cormorant Italic、letter-spacing 5）
- 四方の点: `#C9A16A`、r=2.5

---

### B · 家紋（代替案）

六角形に鍋のモチーフ。「家」と「料理」の両方を表現。  
アイコン単体での視認性が高く、アプリアイコンとして特に有効。

**SVG ファイル:** `docs/brand/svg/logo-crest.svg`

---

### C · 縦組み

縦書きの書籍ロゴ。スプラッシュ画面・オープニング演出向き。  
アイコンには不向きだが、ブランドの世界観を強く表現できる。

**SVG ファイル:** `docs/brand/svg/logo-vertical.svg`

---

### D · 横組みワードマーク

`臺所` + `D A I D O K O` の横型レイアウト。  
ヘッダーバーを持たない現行の UI 設計では画面内常時表示には使用しない。  
ストアのスクリーンショット・プロモーション素材・印刷物に転用する。

**SVG ファイル:** `docs/brand/svg/logo-wordmark.svg`

## 6. 実装上の注意

- **臺（正字体）のフォント対応:** Yu Mincho（Windows）/ Hiragino Mincho ProN（macOS / iOS）では表示可能。Android では Noto Serif JP でフォールバックが発生する場合あり。アプリバンドルへのフォント同梱を推奨。
- **最小表示サイズ:** 蔵書印コンセプトは 32×32px 以上で使用すること。それ以下では 臺所 の字形が潰れる。
- **ダークモード専用:** 現バージョンはダークモード専用デザイン（`#0A0805` 背景前提）。ライトモード対応時は別途カラー定義が必要。
