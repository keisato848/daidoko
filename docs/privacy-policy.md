# だいどこ プライバシーポリシー

最終更新日: 2026-06-03

「だいどこ」（以下「本アプリ」）は、家庭内で利用するレシピ手帳アプリです。本ポリシーは、本アプリがどのような情報を取り扱うかを説明するものです。

## 1. 本アプリが取り扱う情報

本アプリでは、ユーザーが入力または作成した以下の情報を端末内に保存します。

- レシピ情報（タイトル、材料、手順、メモ、タグなど）
- 調理記録（評価、メモ、調理日時など）
- 料理写真、OCR 取り込み元テキスト、写真取り込み元情報
- 家族グループ名、表示名、招待コードなどのアプリ内設定情報
- ローカルバックアップおよび移行用バックアップに含まれるデータ

これらの情報は、ユーザーがレシピの保存、閲覧、編集、バックアップ、復元を行うために利用されます。

## 2. 権限の利用目的

本アプリは、以下の権限をユーザーの明示操作時にのみ利用します。

- カメラ: レシピ写真や調理記録写真の撮影、OCR 取り込みのため
- 写真ライブラリ: 端末内の画像を選択し、レシピ作成や調理記録に利用するため
- ファイル選択: 移行用バックアップファイルを選択して復元するため

## 3. 外部送信される情報

### 3.1 URL 取り込み機能

ユーザーが「URL から取り込み」機能を使う場合、入力した URL は本アプリのサーバーに送信されます。サーバーは、その URL のページを取得してレシピ情報を抽出し、下書きデータを端末へ返します。

この処理は、URL 取り込み機能を提供する目的のためにのみ行います。

### 3.2 共有機能

ユーザーが共有操作を行った場合、本アプリは以下の情報を OS の共有機能に引き渡します。

- 家族グループの招待コード
- ユーザーが作成した移行用バックアップファイル

共有先となるアプリやサービスでの取り扱いは、各サービスのプライバシーポリシーに従います。

## 4. 外部送信しない情報

現在のアプリ公開版では、以下を目的とした SDK または機能は実装していません。

- 広告配信
- アプリ利用状況の解析やトラッキング
- 行動ターゲティング広告

## 5. 保存場所と保持期間

- レシピ、メモ、写真、バックアップなどのアプリデータは、主にユーザーの端末内に保存されます。
- ユーザーが削除しない限り、これらのデータは端末内に保持されます。
- アプリをアンインストールした場合、端末内のアプリデータは OS の仕様に従って削除されることがあります。
- URL 取り込み機能の利用時には、サーバー側で処理に必要な通信が発生します。運用上必要な範囲で、サーバーの基本的なアクセス記録が残る場合があります。

## 6. 第三者サービス

本アプリは、機能提供のために以下のような外部サービスまたは外部システムと連携する場合があります。

- ユーザーが指定したレシピサイト
- ユーザーが選択した共有先アプリ
- 本アプリのサーバーホスティング環境

## 7. お子様の利用について

本アプリは 13 歳未満の子どもを主たる対象としていません。

## 8. セキュリティ

本アプリは、取り扱う情報の保護に合理的な措置を講じます。ただし、インターネット通信や端末保管には技術的な限界があり、完全な安全性を保証するものではありません。

## 9. 本ポリシーの変更

本ポリシーは、機能追加や法令対応に応じて改定することがあります。重要な変更がある場合は、アプリの掲載情報または関連ページで告知します。
