## 1. デザインコンセプト

**「深夜の私設レシピ蔵書庫」**

シネマティック（ダーク背景・写真が浮かぶ）× アナログカード（紙の質感・書物の章立て）の融合。  
料理アプリ定番の食欲喚起カラー（赤・オレンジ）を避け、長く使っても飽きない佇まいを目指す。

### キーワード

- 奥行き感のある暗さ（ランプ一灯の書斎）
- 手触りのある紙・活字
- 家族の記録が積み重なる厚み
- 世代を超えて引き継げる永続性

## 2. ブランドアイデンティティ

### アプリ名表記

| 用途       | 表記                           |
| ---------- | ------------------------------ |
| 正式名称   | 臺所（正字体）                 |
| 読み       | だいどこ                       |
| ラテン表記 | DAIDOKO                        |
| サブコピー | 家族で育てる、台所のレシピ手帳 |

### ロゴ方針

- **アイコン**: 蔵書印（円形シール）または家紋スタイル（六角形）に `臺所` / `臺` 一文字を配置
- **ヘッダーワードマーク**: `臺所` + `D A I D O K O`（イタリック、広めのレタースペーシング）
- **色**: ゴールド（#C9A16A）on ダーク（#0A0805）を基本とする
- **フォント**: 正字体を使用するため、明朝体フォントのバンドルが必要（Yu Mincho / Noto Serif JP など）

## 4. タイポグラフィ

### フォントスタック

| 役割         | フォントファミリー              | 備考                         |

## 7. コンポーネントガイドライン

### ヘッダー

- 背景: `rgba(10,8,5,0.88)` + `backdrop-filter: blur(20px)`
- ブランド表記: `臺所`（明朝 18px）+ `D A I D O K O`（Cormorant Italic 8px）

### タブバー

- 4タブ: ホーム（一）/ 蔵書（二）/ 記録（三）/ 設定（四）
- 章番号（漢数字）+ 日本語ラベル + ラテン略称の三段組み
- アクティブ: `#D4B078`、非アクティブ: `#7A6450`

### リスト行（レシピ行）

- ホバー時: 背景 `rgba(201,161,106,0.06)` + 左パディング +4px のスライド
- 矢印（`→`）はホバー時のみ不透明度 1 に遷移

### アクションボタン

- プライマリ: 背景 `#C9A16A`、テキスト `#0A0805`、`box-shadow: 0 12px 30px rgba(201,161,106,0.35)`
- セカンダリ: `background: transparent`、`border: 1px solid #C9A16A`、テキスト `#C9A16A`

### 覚書カード

- `border: 1px solid #A07A44`、背景 `rgba(201,161,106,0.03)`
- 左上に `覚書` ラベル（`background: #0A0805` で浮かせる）

### フィルターピル

- デフォルト: `border: 1px solid #2E2418`、テキスト `#C9A16A`
- ホバー: 背景 `#C9A16A`、テキスト `#0A0805`

## 8. 装飾・テクスチャ

### グレイン（ノイズ）

全画面に SVG フィルターで薄いノイズを重ねる（`opacity: 0.5`、`mix-blend-mode: overlay`）。  
実装: `grain` クラスの `::before` 疑似要素に SVG データ URI を適用。

### ランプグロウ

画面上部に `radial-gradient` で暖色の光源効果（`rgba(201,161,106,0.10)`）。  
`flicker` アニメーションで微かに揺らす。

### ヴィネット

画面四隅に `radial-gradient` で暗化（`rgba(0,0,0,0.4)`）。

## 9. 画面別デザイン指針

### ホーム

- ヒーローセクション: 今日の日付（和暦）+ 大見出し + 引用文（覚書）
- 統計ブロック: 3カラムグリッド、数字を大きな Cormorant Italic で表示
- アクティビティリスト: 調理者・料理名・日時の3カラム

### 蔵書一覧

- ヘッダー: 大きな数字（蔵書数）+ タイトル
- フィルターバー: 横スクロール可能なピル群
- レシピ行: 番号（Mono）/ タイトル（明朝）/ カテゴリ・調理回数（Faint）

### レシピ詳細

- ヒーローセクション: NO. + カテゴリ / タイトル（52px）/ ローマ字 / 副題
- メタグリッド: 4カラム（人数・時間・調理回数・最終調理）
- 写真プレースホルダー: `aspect-ratio: 4/3`
- 材料: ドット罫線（`· · · ·`）で名前と分量をつなぐ
- 手順: 番号（38px Cormorant Italic）+ テキスト
- 覚書カード: 引用スタイル

### 追加画面

- 方法選択: 3択（URL / OCR / 手動）を番号付きカードで縦並び
- URL 入力: シンプルな入力欄 + 読み込みボタン + プレビューカード
- 手動入力: セクション分割フォーム（材料・手順は動的追加）

### 料理中モード

- 全画面ダーク背景
- 手順テキスト: 28–32px、読みやすい行間
- フッターに「前へ」「次へ」ボタン
- `keepAwake` API で画面常時 ON

## 10. 実装上の注意

- フォント: Google Fonts CDN は実装時に確認。オフライン対応のためバンドルを推奨
- 正字体（臺）: Yu Mincho では表示可能だが、他フォントではフォールバックが発生する場合あり
- アニメーション: `prefers-reduced-motion` メディアクエリでモーションを制御すること
- 画像: レシピ写真は WebP 形式を優先し、JPEG をフォールバックとする
