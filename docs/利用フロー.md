---

## 1. アプリ起動フロー

```mermaid
flowchart TD
    A([アプリ起動]) --> B{初回起動?}
    B -- Yes --> C[オンボーディング\nアカウント作成 / 家族グループ設定]
    B -- No --> D{ローカルデータあり?}
    C --> E[ホーム画面]
    D -- Yes --> E
    D -- No --> F[空の蔵書\nレシピ追加を促す]
    F --> E
```

---

## 2. レシピ追加フロー

### 2.1 方法選択

```mermaid
flowchart TD
    A([＋ボタン / 追加タブ]) --> B[方法選択画面\n① URL  ② OCR  ③ 手動]
    B --> C1[URL 取り込み]
    B --> C2[OCR 取り込み]
    B --> C3[手動入力]
```

---

### 2.2 URL 取り込みフロー

```mermaid
flowchart TD
    A([URL 入力画面]) --> B[URL 貼り付け]
    B --> C[「読み込む」タップ]
    C --> D{通信成功?}
    D -- No --> E[エラー表示\n再試行 / 手動入力へ]
    D -- Yes --> F{JSON-LD あり?}
    F -- No --> G[フォールバック案内\n手動入力へ誘導]
    F -- Yes --> H[プレビュー表示\nタイトル・材料・手順]
    H --> I{内容を確認}
    I -- 修正あり --> J[フィールド編集]
    J --> I
    I -- OK --> K[「蔵書に追加」タップ]
    K --> L([蔵書一覧へ遷移\n新レシピがハイライト])
```

---

### 2.3 OCR 取り込みフロー

```mermaid
flowchart TD
    A([OCR 画面]) --> B[カメラ起動]
    B --> C[撮影]
    C --> D{画像品質チェック}
    D -- 不良 --> E[再撮影を促す]
    E --> B
    D -- OK --> F[OCR 処理中\nスピナー表示]
    F --> G{認識成功?}
    G -- 失敗 --> H[エラー表示\n手動入力へ誘導]
    G -- 部分成功 --> I[テキスト確認画面\n低信頼箇所をハイライト]
    G -- 成功 --> I
    I --> J[ユーザーが確認・修正]
    J --> K[保存]
    K --> L([蔵書一覧へ])
```

---

### 2.4 手動入力フロー

```mermaid
flowchart TD
    A([手動入力フォーム]) --> B[タイトル入力]
    B --> C[カテゴリ・人数・時間]
    C --> D[写真選択\n任意]
    D --> E[材料を入力\n＋で行追加]
    E --> F[手順を入力\n＋で行追加]
    F --> G[覚書・タグ\n任意]
    G --> H{保存}
    H -- タイトル未入力 --> I[バリデーションエラー]
    I --> B
    H -- 下書き保存 --> J([下書きリストへ])
    H -- 完了 --> K([蔵書一覧へ])
```

## 3. 調理フロー（メインシナリオ）

```mermaid
flowchart TD
    A([ホーム or 蔵書一覧]) --> B[レシピを検索 / 選択]
    B --> C[レシピ詳細画面]
    C --> D{目的は?}
    D -- 材料確認 --> E[買い物モード\n材料チェックリスト]
    D -- 調理開始 --> F[料理中モード\n手順全画面表示]
    E --> F
    F --> G[手順をスワイプで進める]
    G --> H{最後の手順?}
    H -- No --> G
    H -- Yes --> I[「作った！」ボタン]
    I --> J{写真・コメント追加?}
    J -- 追加 --> K[写真撮影 / コメント入力]
    K --> L[記録保存]
    J -- スキップ --> L
    L --> M([ホームの家族タイムラインに反映])
```

---

## 4. 家族共有フロー

### 4.1 招待する側（オーナー）

```mermaid
flowchart TD
    A([設定 → 家族管理]) --> B[「家族を招待」タップ]
    B --> C[QR コード / 招待コード生成\n有効期限 24 時間]
    C --> D{共有方法}
    D -- QR --> E[QR コードを相手に見せる]
    D -- コード --> F[6桁コードを伝える]
    E --> G([相手が参加完了])
    F --> G
    G --> H[メンバー一覧に表示]
```

---

### 4.2 参加する側（家族メンバー）

```mermaid
flowchart TD
    A([設定 → 家族グループに参加]) --> B{参加方法}
    B -- QR スキャン --> C[カメラで QR を読み取る]
    B -- コード入力 --> D[6桁コードを入力]
    C --> E{コード有効?}
    D --> E
    E -- No --> F[エラー: コード無効 / 期限切れ]
    F --> A
    E -- Yes --> G[グループ参加完了]
    G --> H([オーナーの蔵書にアクセス可能になる])
```

## 5. タイムライン・記録閲覧フロー

```mermaid
flowchart TD
    A([記録タブ]) --> B{表示モード}
    B -- タイムライン --> C[家族全員の調理ログ\n新しい順に一覧]
    B -- カレンダー --> D[月別カレンダー\n調理した日がマーク]
    B -- ギャラリー --> E[調理写真一覧\nグリッド表示]
    C --> F[ログをタップ]
    D --> G[日付をタップ → その日のログ一覧]
    E --> H[写真をタップ → 詳細]
    F --> I([レシピ詳細 or ログ詳細へ])
    G --> I
    H --> I
```

## 6. 画面遷移マップ（全体）

```mermaid
flowchart LR
    subgraph Tab1[一 ホーム]
        Home[ホーム画面]
    end

    subgraph Tab2[二 蔵書]
        Library[蔵書一覧]
        Detail[レシピ詳細]
        Cook[料理中モード]
        Shop[買い物モード]
        AddEntry[追加方法選択]
        AddURL[URL 取り込み]
        AddOCR[OCR 取り込み]
        AddManual[手動入力]
        Cooked[作った！記録]
    end

    subgraph Tab3[三 記録]
        Timeline[タイムライン]
        Calendar[カレンダー]
        Gallery[ギャラリー]
    end

    subgraph Tab4[四 設定]
        Settings[設定]
        Family[家族管理]
        Invite[招待画面]
    end

    Home --> Detail
    Home --> Timeline
    Library --> Detail
    Library --> AddEntry
    AddEntry --> AddURL
    AddEntry --> AddOCR
    AddEntry --> AddManual
    Detail --> Cook
    Detail --> Shop
    Detail --> Cooked
    Detail --> AddManual
    Settings --> Family
    Family --> Invite
```
